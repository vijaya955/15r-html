HTML

  • HTML stand for Hypertext Markup Language.
  • HTML was introduced by Tim Berner's Lee in 1991.
  • In 2014 HTML version 5 was introduced.
  • HTML is used to create a base structure for a webpage.
  • In html we have special symbols called Tags.

Heading Tags

  • Heading tags displays title/heading on a webpage.
  • Headings tags are defined with h1 to h6 tags.

Paragraph Tags

p defines a paragraph and it starts on a new line and it contains a block of text.

Types of Tags

1.Void Tags

  1. It is also called as self closing tags.
  2. break and horizontal lines are void tags.

2.anchor Tags

  • anchor tag is used to create a link.
  • If we want to add an additional information to an element we use attributes.

    • Block Line elements

    Inline elements

    Formatting Tags

    It is used to style or structure the particular text or content of website.
    • bold: it speifies the boldness for a text.
    • italic: it specifies the italic styled text.
    • strong: defines the text with strong importance.
    • emphasize: it specifies emphasized text.
    • underline: it underlines a text.
    • subscript: specifies subscript text
    • superscript: specifies superscripted text.
    • deleted text: represent a range of text that has been deleted from a document.
    • mark text: it highlights some part of text.
    • blockquote: it represent text that is quoted from other website.
    • abbr: defines an abbreviation.
    • pre: defines preformatted text.
    • cite: defines title of a creative work.
    • address: define address
    • code: defines a piece of computer code.
    • keyboard: defines keyboard input

    Tables

    • It used to represent data in tabular format.
    • Tables are defined with table tag.
    • Rows are represented with tr.
    • Columns are represented with td.
    • Column headings are represented with th.

    image tag

    It is used to specify image in html document.It is a empty tag and its a inline element.

    anchor tag

    It defines a hyperlink which is used to link from one page to another.
    1. Unvisited link is underlined and blue.
    2. Visited link is underlined and purple.
    3. Active link is underlined and red.

    Strucuture of URL

    • It is a technique that is utilized by browsers to get the resources that are published on the web.
    • The components of URL consists of subdomain,domain,toplevel domain.
    • http port number is 80.
    • https port number is 443.
    • Top level domain represent the category.

    List tag

    Lists allow web developers to group a set of related items in lists.

      Ordered list

    • Its is used to specify list items in sequence manner.
    • It is defined with ol tags.
    • List items are defined with li.

      • Unordered list

    • It is used to specify list items in particular order.
    • It is defined with ul tags.

      • Description list

    • It is used to create description items in html document.
    • It is defined with dl tags.
    • It has 2 types:
    • Description term : defined with dt
    • Description data : defined with dd
