Visual Studio Code(VS):

It is an IDE(Integrated Development Environment) software which combines with graphical user interface.

Features of VS Code:

-platform independent
-development capabilities
-open source
-debugging
-command palette
-command line

Shortcut Keys:

Keys
Description
ctrl+s to save the file
ctrl+c to copy the content
ctrl+v to paste the content
ctrl+x to cut the selected text
alt up to move a line up
alt down to move a line down

Emmets :

Emmet Description
! used for html structure
> used for child tag
* used for multiple elements
+ used for sibling tag
^ it adds sibling to the parent
